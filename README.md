# LicitaHub

Plataforma para rede empresarial de engenharia consultiva, central de editais e match para formacao de parcerias e consorcios.

## Stack definida

- Frontend: React com HTML e CSS
- Backend: Go
- Banco de dados: PostgreSQL

## Estrutura inicial

- `src/`: frontend React
- `backend/`: API inicial em Go
- `database/`: ponto inicial para scripts/migracoes
- `preview-react-cdn.html`: preview navegavel temporario sem build local

## Observacao

O preview por CDN existe para validacao visual enquanto o ambiente local de dependencias npm e ajustado.
